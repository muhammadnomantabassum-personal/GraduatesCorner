# Graduates Corner SEO Launch Plan

This playbook covers the operational work that cannot be completed only by deploying code. Rankings are earned through crawlable, useful content, trusted mentions, accurate listings, and sustained measurement. They cannot be guaranteed.

## 1. Search Engine Setup

1. Create a Google Search Console Domain property for `graduatescorner.com` and verify it with the DNS record Google provides.
2. Add the supplied HTML verification token to `GOOGLE_SITE_VERIFICATION` in Vercel and redeploy.
3. Submit `https://graduatescorner.com/sitemap.xml` in Search Console.
4. Inspect the homepage, the three listing pages, and several live opportunity URLs. Request indexing after validation.
5. Create a Bing Webmaster Tools property, set `BING_SITE_VERIFICATION`, submit the sitemap, and enable IndexNow when credentials are available.
6. Monitor Coverage, Page indexing, Core Web Vitals, HTTPS, Manual actions, and Job posting enhancement reports weekly.

## 2. Google Jobs Operations

- Keep `JobPosting` data only on active, complete vacancy detail pages.
- Every vacancy must display the same title, organization, description, location, deadline, and application destination that appears in its schema.
- Remove structured data when a deadline passes. The application does this automatically.
- Reject fake, incomplete, duplicate, pay-to-apply, or keyword-stuffed listings.
- Use Google's Indexing API for newly published, changed, and expired job URLs after a dedicated Google Cloud service account is approved. Never expose its credential in browser code.

## 3. Legitimate Off-Page Authority

- Ask each verified university or company to link its Graduates Corner organization or opportunity page from the institution's careers, research, student services, or partners page.
- Publish original quarterly reports using aggregated platform data, such as doctoral hiring fields, application deadlines, funded-position trends, and graduate-program locations. Pitch the findings to university career offices and education journalists.
- Build relationships with doctoral schools, graduate associations, Erasmus networks, student unions, research groups, career centres, and international offices.
- Maintain complete, consistent organization profiles on LinkedIn, relevant startup or education directories, and professional association directories.
- Contribute expert articles and webinars about PhD applications, thesis collaboration, and graduate recruitment. Link only where editorially relevant.
- Provide a press and partner page with a concise description, logo assets, official URLs, contact information, and attribution guidance.

Do not buy backlinks, automate directory submissions, exchange links at scale, or use keyword-heavy guest-post networks. These practices can trigger ranking suppression or manual actions.

## 4. Content Programme

Publish one high-quality search-intent article each week. Every article should answer a real user question, cite reliable sources, include a named author, show a publication date, and link to relevant live opportunities.

Priority topic clusters:

- PhD positions by country and research field
- How to apply for funded PhD positions
- Academic CV and motivation-letter guidance
- Master's thesis positions in industry
- Graduate trainee programme application timelines
- International applicant documentation and interview preparation

Avoid creating thin city or country pages that repeat the same text. Create a location page only when it has unique listings, useful local guidance, and enough demand.

## 5. Measurement

Track these monthly:

- Indexed valid pages versus submitted sitemap pages
- Non-branded impressions and clicks by query cluster
- Average position and click-through rate for priority landing pages
- Google Jobs impressions and application clicks
- Referring domains earned from universities, companies, and editorial publications
- Organic registrations, wishlists, and outbound application clicks
- Core Web Vitals and mobile usability

Review pages with high impressions but weak click-through rates first. Improve their titles and descriptions without changing URLs unnecessarily.
